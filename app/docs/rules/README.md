## Résumé ultra-rapide (pour la VM)

- **Attributs** : Corps, Cœur, Esprit (valeur 1–9). SR = 20 − attribut.
- **Stats dérivées** : Endurance = Corps + bonus culture, Espoir = Cœur + bonus culture, Parade = Esprit + bonus culture.
- **Jet de base** : 1 dé du Destin (d12) + N dés de Succès (d6) ≥ TN (cible, défaut 14).
- **Épuisé** : Endurance courante < Charge totale (équipement + Fatigue).
- **Mélancolique** : Espoir courant < Ombre accumulée.
- **Blessé** : toggle manuel après coup perforant réussi.
- **Sagesse/Vaillance** : rangs 1–6, 1 Vertu/Récompense par rang à partir du rang 2.
