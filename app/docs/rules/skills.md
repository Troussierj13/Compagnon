# Compétences — The One Ring 2e

## Structure générale

- **18 compétences communes** réparties en 3 groupes (6 par attribut).
- **4 compétences de combat** (armes), indépendantes.
- Chaque compétence a un **rang** de 0 à 6.
- Le rang = nombre de **Dés de Succès** lancés pour ce jet.
- Une compétence peut être **favorisée** (2 Dés du Destin, garder le meilleur).

- Le rang (0–6) s'applique exactement comme les compétences communes : = Dés de Succès lors d'une attaque.
- **Chaque culture** donne un rang de départ dans 1 à 2 compétences de combat.

## Compétences favorisées

- Une compétence **favorisée** donne l'avantage sur le jet : 2 Dés du Destin, garder le meilleur.
- Chaque culture attribue **2 compétences favorisées** de base à la création.
- La vocation peut en ajouter davantage.
- La Vertu "Maîtrise" accorde **2 compétences communes favorisées** supplémentaires au choix.

---

## Coût de progression

Compétences communes et de combat utilisent toutes les **Points de Progression** avec le barème universel :

| Passage de rang | Coût |
|---|---|
| 0 → 1 | 4 pts |
| 1 → 2 | 8 pts |
| 2 → 3 | 12 pts |
| 3 → 4 | 20 pts |
| 4 → 5 | 26 pts |
| 5 → 6 | 30 pts |

> Exemple : passer de rang 2 à rang 3 coûte **12 points de Progression**.
> Exemple : passer de rang 0 à rang 2 coûte 4 + 8 = **12 points de Progression**.
> Voir barème complet dans `progression.md`.

---

## Jet avec rang 0

Quand le rang est 0, le personnage n'a aucun Dé de Succès.
Il lance uniquement le **Dé du Destin** et compare au TN.
- Succès uniquement sur Rune de Gandalf ou si le dé seul atteint le TN.
- Risque élevé, mais possible.
